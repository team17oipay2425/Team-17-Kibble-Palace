Final System Proposal

Solution: Discreet dual IR beam counter with a dedicated Dashboard. QR Code form for country of origin tracking.

Our final proposal represents a comprehensive footfall tracking system specifically designed for heritage environments like the Kibble Palace. The solution balances technological sophistication with heritage sensitivity, ensuring accurate data collection without compromising the building's historic character or visitor experience.

System Components

Core Features:

Low-cost microcontroller (e.g. ESP32)

Counts are time-stamped and stored

Tracking both exit and entry records

Visualised via web dashboard (Daily, Weekly, Event-based)

Why It Matters

This solution enables better resource planning, helps justify staffing, and supports long-term funding decisions.

Beyond immediate operational benefits, our system demonstrates how heritage sites can embrace modern technology responsibly. The privacy-first approach and heritage-sensitive installation create a replicable model for other historic venues facing similar challenges.

By providing accurate, real-time visitor data, Glasgow City Council can optimize operations, improve visitor experiences, and make evidence-based decisions about the future of this magnificent Victorian glasshouse.
